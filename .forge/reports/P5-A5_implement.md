## Summary

Added a `hw-probe` CLI subcommand to the `anvilml` binary that calls `detect_all_devices()` from `anvilml-hardware` and prints the resulting `HardwareInfo` as pretty-printed JSON to stdout, then exits 0. The `Cli` struct was restructured to include an optional `#[command(subcommand)]` field, enabling non-server operations without requiring separate binary targets. An integration test verifies the subcommand appears in help output.
