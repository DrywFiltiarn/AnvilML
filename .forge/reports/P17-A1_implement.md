## Summary

Implemented single-job deletion support for the AnvilML server. Added `delete_for_job()` to the artifact store that deletes on-disk PNG files and artifact DB rows for a given job. Added `delete_job()` handler that validates the job is in a terminal state (rejects Queued/Running with 409), calls artifact deletion, then removes the job row from SQLite, returning 204 No Content. Wired the `DELETE /v1/jobs/{id}` route in `build_router()`. Added 5 unit tests covering all code paths.
