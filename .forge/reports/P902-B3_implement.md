The change required three file modifications: workspace dependency declaration, server crate dependency, and router layer insertion. All 215 workspace tests pass, all four platform cross-checks pass (Linux mock, Windows cross, Linux real, Windows cross), clippy is clean, format check is clean, and the config surface sync gate passes.
