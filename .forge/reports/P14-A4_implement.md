## Summary

Implemented the `GET /v1/artifacts/{hash}` endpoint that serves stored PNG artifacts by their content-addressed SHA-256 hash. Added `get_path()` method to `ArtifactStore` for path resolution, created a new `handlers/artifacts` module with the `serve_artifact` handler, wired the route in `lib.rs`, updated module exports, and added three integration tests covering 404, 200 with headers, and correct byte content. Bumped `anvilml-server` version from 0.1.6 to 0.1.7 and added the `fs` feature to `tower-http` in `Cargo.toml`.
